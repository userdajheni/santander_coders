Listas ordenadas e listas não ordenadas



Listas não ordenadas

São listas que não seguem ordem, ou seja, ao invés de serem enumeradas elas seguem pontos para destacar os itens. A tag ul define uma lista não ordenada e a tag li define um item da lista:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

Listas ordenadas

São listas que seguem uma ordem. A tag que define uma lista ordenada é ol:

  1. Item A
  2. Item B
  3. Item C
  4. Item D
