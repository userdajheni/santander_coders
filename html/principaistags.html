Principais tags



Imagens

src é onde especificamos o caminho da imagem, pode ser link externo ou interno

alt serve para inserir um texto que aparecerá caso a imagem não seja carregada

Podemos também na própria tag img inserir os atributos width e heigth para definir largura e altura exata que deseja.



Tags de uso genérico

Uma tag de uso genérico é aquela que pode ser usada para qualquer conteúdo. Essas tags são: span e div.

Usamos o span para poder estilizar elementos. Ela é do tipo inline o que significa que não quebra linha. Exemplo: Lorem ipsum dolor sit estou dentro de um span adipisicing elit. Similique animi estou dentro de um span hic blanditiis quibusdam.

O div é uma tag do tipo block, ou seja, ela faz quebra de linha por ser um bloco. Exemplo: Lorem ipsum dolor sit

estou dentro de uma div
adipisicing elit. Similique animi
estou dentro de uma div
hic blanditiis quibusdam.
